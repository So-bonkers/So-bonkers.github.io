Research

Accepted IEEE WCNC 2026

Detecting Convolutional Codes via a Markovian Statistic

Shubhankar Kamthankar, Guneesh Vats, Arti Yardi — IIIT Hyderabad

A novel Markov chain framework for blind detection of convolutional codes in noisy intercepted data. The Viterbi metric sequence is modelled as a Markov chain; code detection becomes hypothesis testing between two MCs. Provides closed-form asymptotic guarantees via error exponents. Outperforms the syndrome-former baseline — Perr ≈ 0 vs 0.41 at N = 500, p = 0.2.

4th Place PAN @ CLEF 2026

Multi-Encoder Ensemble with Sequential BiLSTM Refinement for Style Change Detection

Shubhankar Kamthankar, Arti Yardi — IIIT Hyderabad

Four-stage sentence-level authorship-change detection: CoSENT fine-tuning of DeBERTa-v3, RoBERTa-large, and MPNet-base; difficulty-adaptive MLP classifiers with Focal BCE + R-Drop; Optuna ensemble fusion; document-level BiLSTM refinement over full boundary sequences.
