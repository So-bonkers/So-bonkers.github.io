Research

Publications

IEEE WCNC 2026 · Accepted

Detecting Convolutional Codes via a Markovian Statistic

Shubhankar Kamthankar, Guneesh Vats, Arti Yardi

IEEE Wireless Communications and Networking Conference 2026

A novel Markov chain-based framework for detecting convolutional codes in noisy intercepted data. We decode received data via the Viterbi algorithm and model the resulting metric sequence as a Markov chain, reducing code detection to hypothesis testing between two MCs. Provides analytical asymptotic guarantees via error exponents and outperforms the syndrome-former baseline — e.g. achieving Perr ≈ 0 vs 0.41 at N=500, p=0.2.

Method: Viterbi + Markov hypothesis test Channel: BSC
PAN @ CLEF 2026 · 4th Place

Multi-Encoder Ensemble with Sequential BiLSTM Refinement for Style Change Detection

Shubhankar Kamthankar, Arti Yardi

CLEF 2026 Working Notes — PAN Multi-Author Writing Style Analysis Shared Task

A four-stage sentence-level authorship-change detection pipeline: (1) CoSENT fine-tuning of DeBERTa-v3, RoBERTa-large, and MPNet-base for style-sensitive embeddings; (2) difficulty-adaptive MLP classifiers with Focal BCE + R-Drop; (3) Optuna-optimised ensemble fusion; (4) BiLSTM document-level refinement over complete boundary sequences.
